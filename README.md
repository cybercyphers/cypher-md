**Installation:**

###### clone the repository
```
git clone https://github.com/cybercyphers/cypher-md.git
```
###### change directory into the cloned repo
```
cd cypher-md
```
###### read comments and go according to it
```
nano config.json 
```
###### save file 
---

###### install dependencies and start
```
npm install && npm start
```
